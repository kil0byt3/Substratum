1. Simply run ServerStart.bat to start the server and download Forge.

2. Minecraft Forge will be installed automatically the first time you run ServerStart.bat.

3. The server will then close. Update eula.txt to continue, agreeing to the Mojang ToS by changing "false" to "true". 

4. Run ServerStart.bat to start the server.





Note: It is safe to remove the Forge Installer jar once the installation has finished.
